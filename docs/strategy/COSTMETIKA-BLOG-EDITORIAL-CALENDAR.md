## SEO Strategy

### Keyword Research Approach
1. Use Google Keyword Planner, Ahrefs, or SEMrush for search volume
2. Target long-tail keywords (3-5 words) with lower competition
3. Focus on educational intent (how-to, what is, guide, etc.)
4. Include location-based keywords where relevant

### On-Page SEO Checklist (Every Post)
- [ ] Primary keyword in title (H1)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Secondary keywords naturally throughout
- [ ] Meta description 150-160 characters with keyword
- [ ] Image alt text with descriptive keywords
- [ ] Internal links to 3-5 related posts/products
- [ ] External links to 1-2 authoritative sources (studies, dermatology sites)
- [ ] URL slug includes primary keyword
- [ ] 1,500-2,500 word count (longer for pillar content)
